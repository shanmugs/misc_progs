# jOOQ Example with MySQL

## Guide
https://hellokoding.com/jooq-example-with-mysql/

## What you'll need
- JDK 1.7 or later
- Maven 3 or later
- MySQL 5.6 or later

## Stack
- Java
- jOOQ
- MySQL

## Run
`mvn clean compile exec:java -Dexec.mainClass=com.hellokoding.jooq.Application -Dexec.cleanupDaemonThreads=false`
